ABOUT ME

I am a curious person who found in software development a way to create new things and, at the same time, help people by developing useful applications.

Aside from developing websites and web applications, I'm also passionate about understanding the physical world around us. That's why I'm enrolled in an undergraduate degree in Physics (with minor in Computer Science) at the Universidade do Porto, Portugal. Therefore, when I'm not creating new web projects, I'm probably learning about diferential calculus, electromagnetism, quantum mechanics or planetary astronomy!

I have more than a year of experience developing web applications using the following technologies:

  • HTML5
  • CSS3
  • JavaScript
  • JQuery
  • PHP
  • MySQL

I have a basic understanding of Node.Js and Express. I intend to create a few other projects using this technology in a near future. I also have experience with the Python programming language and a few of its libraries for data analysis: numpy, matplotlib, among others. I also have experience with the PHP framework Laravel, which I used intensively for a few months.

One of the things I enjoy the most is to learn new things. And one of the awesome things about the web development industry is that you get to do exactly that every day! Even though I never dealt with technologies such as Ruby on Rails, Python web frameworks (Django and Flask), the asp.Net technologies and so many others, I do have the desire to learn them if necessary. I believe that all these technologies have the same goal: to help developers create useful products. And I plan to do so, regardless of the technology or stack used.
